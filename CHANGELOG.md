# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-01

### Added
- Initial release of pw-foundation
- BasePage class for page object patterns
- BaseAPI class for API testing
- Custom fixtures (auth, api, data, logger)
- Configuration factory with environment support
- Comprehensive test suite
- CI/CD pipeline with GitHub Actions
- Complete documentation suite
- TypeScript type definitions
- Utility classes (DataFactory, FileUtils, Logger, WaitHelpers)

### Features
- Type-safe page object creation with locator organization
- REST API testing with automatic response typing
- Test data generation with Faker.js integration
- Environment-based configuration management
- Custom reporters and logging utilities
- Parallel test execution support
- Comprehensive error handling and assertions

### Documentation
- Getting Started guide
- Page Object Guide
- API Testing Guide
- Playwright Best Practices
- TypeScript Best Practices
- Contributing guidelines
